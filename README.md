# MHSDS Synthetic Data Warehouse

A portfolio project modelling a simplified NHS MHSDS-inspired EPR-to-reporting workflow for mental health services, built with:

- SQL Server 2022 running in Docker
- VS Code + SQL server extension
- T-SQL
- Python 3
- Git/GitHub
- Markdown documentation


> ⚠️ **Disclaimer**
> This project uses **100% synthetic data only** and no real patient information is used.


## Project structure

```text
mhsds-synthetic-data-warehouse/
├── docs/                 Project documentation and data dictionary
├── sql/                  Database, schema, views, checks, KPIs, and procedures
├── python/               Synthetic data generation
├── dashboard/            Dashboard requirements and example queries
└── project_management/   Planning and task tracking
```
